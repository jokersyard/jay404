# jay404

This is my Portfolio where I keep a track of my projects and work-related assignments.

## Latest Updates

- 19-06: Adding new UI/UX improvements
- 19-06: Performance optimization for components
- 19-06: Enhanced metadata and SEO configuration

## Tech Stack

- **Framework**: Next.js 14.0.4
- **UI Library**: React 18.2.0
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Heroicons
- **Analytics**: Vercel Analytics

## Getting Started

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see your portfolio.

## Build & Deploy

```bash
npm run build
npm start
```
